# リリースノート：バージョン 0.97.1 から 0.97.2 の変更点

## 主な変更点
言語システムにおける大きな変更点はありません。

## 既存のワードに対する変更点
型情報に関するワードが変更されています。
既存のワードの名前を変更し、新たに同名のワードを追加するなど
複雑な状況になっています。
詳しくは次の名称変更についての説明をご覧ください。

### 名称変更
これまで TOS の値についての型情報を取り出すワードは `@type>`
でしたが、型情報への変換（to type）なので `@>type` へとワー
ド名を変更しました。

上記に関連して、TOS を消費するタイプのワードとして `>type` というワードを新規に追加しました。
もともと `>type` というワードは存在していましたが、
こちらも名称変更する必要がありましたので変更しました。

キャストを行うワードは `>type` という名前でしたが、これは型（type）から値を生成する
（キャストする）という考えに基づけば `type>` (from type) となるべきです。
この考えに基づき名前を変更しました（`>type` から `type>` へ変更）。

## 新たに追加されたワード
`>type` というワードを名称変更し、同名の `>type` というワードを新しく追加しました。

## 廃止されたワード
廃止されたワードはありません。

## 処理系 para について
特に変更はありません。

## 同梱されるサンプルプログラムについて
型情報を用いるサンプルプログラム fastInvSqrt.p8e が新しいワード名に対応するよう変更されています。

## その他の変更
ありません。
